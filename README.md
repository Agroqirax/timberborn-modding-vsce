# Timberborn modding tools

![banner](https://shared.cloudflare.steamstatic.com/store_item_assets/steam/apps/1062090/header_alt_assets_5.jpg?t=1726750499)

This project helps modders with creating mods for [timberborn](https://store.steampowered.com/app/1062090/Timberborn/).

## Installation

Install using vscode

```shell
code --install-extension agroqirax.timberborn-modding-ext
```

## Features

- ✅ Json validation for specifications
- ✏️ Json autocomplete for specifications
- ✍️ Json, C# & TAPI snippets
- 🚀 Launch game command & custom launch config
- 📖 Quick links to documentation & custom links
